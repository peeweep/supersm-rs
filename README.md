# supersm-rs

This project is a dotfiles manager, as a rust impl for [supersm](https://github.com/peeweep/supersm).


```
Usage: supersm-rs [OPTIONS]

Options:
  -a, --add <ADD>
  -d, --delete <DELETE>
  -t, --target <TARGET>  [default: ../]
  -h, --help             Print help
  -V, --version          Print version
```
